Installation

You can install the development version of chromr from GitHub with:

# install.packages("devtools")
devtools::install_github("BradyAJohnston/chromr")

Example

This is a basic example which shows you how to solve a common problem:

library(chromr)
## basic example code
fl <- system.file(
  "extdata",
  "sec.txt", 
  package = "chromr"
)

fl |> 
  chrom_read_quadtech() |> 
  chrom_plot()
#> Warning: One or more parsing issues, see `problems()` for

fl <- system.file(
